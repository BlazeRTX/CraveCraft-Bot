CraveCraft Bot
A Minecraft‑flavored chaos machine that roasts everything you type.
CraveCraft reacts with mood swings, emotional‑damage roasts, villager‑mode nonsense, and Crave‑style humor — all running in a clean HTML/JS chat UI.

🔥 Features
• 	Crave‑style roast engine (mild → emotional damage)
• 	Mood system (Chaotic, Sarcastic, Emotional Damage, NPC Villager, Soft Roast)
• 	Voice‑line generator for extra chaos
• 	Minecraft‑themed insults & metaphors
• 	Intent‑based replies (boredom, building, challenges, survival, biomes, loneliness, school, sleep, etc.)
• 	Typing animation + clean UI
• 	Optional Minecraft sound effects (XP, click, hurt, villager)
• 	100% client‑side — no backend needed
